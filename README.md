# HR Analytics Dashboard (Power BI)

## Project Overview
This project analyzes employee attrition data to identify key workforce trends and drivers of attrition using Power BI.

##  Dashboard Highlights
- Total Employees: 1,470
- Attrition Count: 237
- Attrition Rate: 16.1%
- Average Age: 36.9
- Average Salary: 6.5K
- Average Years in Company: 7.0
